## Object class Method
업캐스팅 된경우 자식 클래스의 메소드를 사용할수 없다.<br>
따라서 자식 클래스의 메소드를 사용하려면 Object class 의 method를 @override 하거나<br>
추상클래스를 따로 만들어 상속한뒤 @override 하여 사용이 가능하다.<br>
